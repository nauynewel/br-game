# 后室：生存模拟器 | Backrooms: Survival Simulator

> 基于 [Backrooms 中文 Wiki](https://backrooms-wiki-cn.wikidot.com/) 的文字生存模拟游戏

🎮 **[点击开始游戏](https://nauynewel.github.io/br-game/)**

## 游戏特性

- 🏚️ **100个层级** — 数据来自 Backrooms 中文 Wiki，包含真实的出入口关系
- 🎲 **846个层级专属事件** — 基于每层 Wiki 内容生成（如 Level 0 的红色墙纸房间、马尼拉房间）
- 📦 **42种 Wiki 物品** — 杏仁水、层级钥匙、命运之骰等
- 🎒 **背包系统** — 最多携带8件物品，探索时随机发现
- 🧠 **双属性系统** — 理智 + 体力，归零即死亡
- 🔍 **探索机制** — 消耗体力探索当前层级，发现特殊事件和物品

## 玩法说明

- **理智归零** = 精神崩溃，游戏结束
- **体力归零** = 体力耗尽，游戏结束
- **探索** = 消耗少量体力，可能触发事件或发现物品
- **休息** = 恢复体力但消耗理智，有概率遭遇危险
- **前往其他层级** = 根据 Wiki 出入口关系移动，消耗取决于目标层级危险度

## 技术

纯前端单 HTML 文件，无需服务器，无依赖。浏览器打开即玩。
